Features:

>> Face Recognition Login System:
  • Instant Face Login: Employees login using AI-powered face recognition (face-api.js).
  • Live Detection: Prevents login using images or videos.
  • Fallback Login: Secure email + password method in case of camera issues.
  • Face Descriptor Storage: Encrypted face-data stored for accurate matching.
>> Smart Location Tracking:
  • Auto Location Capture: Location recorded at login, break-start, break-end, and logout.
  • Live Tracking: Admin can view live employee locations during working hours.
  • Offline Support: If internet is off, system stores location locally and syncs when back online.
  • Geo Validation: Option to restrict attendance to office radius.
>> Attendance & Break Automation:
  • Auto In-Out: Time-in captured immediately after face-login.
  • Break-In/Break-Out: Separate buttons for break tracking with real-time monitoring.
  • Auto Logout Alerts: Reminder if an employee forgets to logout.
  • Daily Summary: Total work duration, total break time automatically calculated.
>> Admin Controls:
  • Approve New Employees: Admin can approve/reject registrations (double authentication).
  • Manual User Creation: Admin can create employees directly without registration.
  • Attendance Filters: Filter by employee name, date range, or status.
  • ZIP Export: Attendance reports can be downloaded as ZIP (CSV/JSON).
  • Live Dashboard: Todayâ€™s attendance, live employees, pending requests, alerts.
>> Employee Panel:
  • Customizable Home Feed: Admin can post announcements that show on employee dashboard.
